# Vue -  Food電商 

## 步驟
1. 為專案加入樣式
  - npm導入bootstrap、bootstrap icons、Vue axios
  - 修改變數
  - Bootstap icon

2. 建立主要路由
  - 首頁
  - 產品列表
  - 單一產品(詳細資訊)
  - Navbar, footer

3. 版型至Vue CLI

4. 頁面製作
  - 首頁
  - 產品列表
  - 單一產品(詳細資訊)
  - Navbar, footer
  - 購物車
  - 訂單 (購物車內容發送後)
